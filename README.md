# 🛒 E-Commerce Customer Behaviour Analysis

## 📌 Project Overview

This project analyses 250,000 e-commerce customer transactions spanning 2020–2023 to uncover what drives purchasing behaviour, where revenue is being lost, and which customer segments represent the highest business value.

The data was sourced from Kaggle and processed through a three-stage Python pipeline — extraction, cleaning, and feature engineering — before being connected to Tableau to build an interactive dashboard with nine analytical views covering revenue trends, churn, return rates, payment behaviour, and customer spending patterns by age group.

Key findings include a 29% revenue drop in 2023, a uniform ~40% return rate across all product categories, a ~20% churn rate with no active retention strategy, and a clear pattern of older customers (56–70) spending significantly more per transaction than younger segments. Books and Clothing together drive ~60% of total revenue, while Electronics and Home remain underperforming despite their higher price points.

The project delivers 10 business insights written in decision language, 5 prioritised recommendations with estimated impact, and a fully interactive Tableau dashboard — giving operations managers and business analysts a clear, data-backed view of where to act.
